# 删除智能场景分析功能计划

## 分析现状

当前代码中包含以下智能场景分析相关功能：

1. `segment_video_scenes` 方法 - 主入口，支持智能和传统场景分割
2. `intelligent_scene_segmentation` 方法 - 基于qwen-plus的智能场景分割
3. `_build_intelligent_segmentation_prompt_with_content` 方法 - 构建智能分割提示词
4. `_parse_intelligent_segmentation_result` 方法 - 解析智能分割结果
5. `traditional_scene_segmentation` 方法 - 传统场景分割

## 保留功能

* `create_scenes_from_slices` 方法 - 根据视频切片创建场景（用户需要的功能）

* `optimize_json_prompt` 方法 - 优化JSON提示词

* `combine_prompts` 方法 - 结合多个模型的prompt

* `generate_video_prompt_for_scene` 方法 - 为单个场景生成提示词

## 删除计划

1. 删除 `segment_video_scenes` 方法
2. 删除 `intelligent_scene_segmentation` 方法
3. 删除 `_build_intelligent_segmentation_prompt_with_content` 方法
4. 删除 `_parse_intelligent_segmentation_result` 方法
5. 删除 `traditional_scene_segmentation` 方法（如果存在）

## 实施步骤

1. 检查并确认所有智能场景分析相关方法
2. 按顺序删除这些方法
3. 确保保留的方法不依赖于删除的方法
4. 验证代码结构完整性
